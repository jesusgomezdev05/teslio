Inicializar git
git init

Iniciar sección (usuario y mail en GitHub)
git config --global user.name "user"
git config --global user.email "mail"

Clonar repositorio
git clone https://github.com/Juanma0247/teslio.git
code teslio

Verificación
git status

Simpre antes de iniciar
git pull origin main

Siempre al terminar
git add .
git commit -m "Aquí di cambios hiciste."
git push



Configurar el hábito:
git pull (al empezar)
git push (al terminar)
VS Code lo hace con botones:
Sync Changes
Pull / Push
